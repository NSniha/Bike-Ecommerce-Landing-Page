# Bike Landing Page

A modern, responsive, and visually engaging bike landing page built with **HTML**, **Tailwind CSS**, **custom CSS**, and **JavaScript**.  
This project is designed to showcase premium bicycles, featured products, categories, accessories, and a polished brand-focused user experience with smooth animations and mobile-first responsiveness.

---

## Overview

The Bike Landing Page is a complete front-end website concept for a bicycle and cycling accessories brand.  
It focuses on clean visual presentation, strong product promotion, smooth section flow, and a premium dark-themed user interface.

The website includes a professionally structured layout with:

- Responsive header and navigation
- Hero section with promotional composition
- Features and facilities section
- Best sellers showcase
- Product categories section
- Featured products section
- Accessories and gear section
- Newsletter and mega footer
- Interactive mobile menu
- Mobile show more / show less product controls
- Smooth reveal animations and hover effects

This project is fully suitable for:

- portfolio presentation
- client delivery demo
- front-end practice
- product landing page reference
- eCommerce landing page inspiration

---

## Live Project Goals

This project was built to achieve the following goals:

- Create a modern and premium bicycle landing page
- Maintain a fully responsive layout for desktop, tablet, and mobile devices
- Present products in a clear, attractive, and conversion-focused structure
- Deliver a smooth browsing experience with animations and interactions
- Keep the code organized, editable, and easy to expand in the future

## Animations and Interactions

The site includes subtle UI animations to improve the user experience without overwhelming the design.

### Included interactions:
- reveal-up animation
- reveal-left animation
- reveal-right animation
- reveal-scale animation
- card hover movement
- icon hover response
- mobile menu open and close interaction
- product toggle interaction for small devices

### Animation behavior:
Animations are triggered using the **Intersection Observer API**, allowing elements to animate when they enter the viewport.
